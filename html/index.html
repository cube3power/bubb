Pure CSS infotip.

Add class "bubb-delay" (or modify global bubb.scss variables) to add a reveal delay. A bubb with HTML content may want to be class(ified) as "bubb-interactive". If you really don't care for transitions add "bubb-still". Alternate anchors with bubb-left or bubb-right.

Attribute-only is in-your-face without animations due to the fact that identical CSS transitions on multiple elements are slightly out of sync.
